1. navigator.onLine

非常简单，但是并不准确-根据MDN的描述：
navigator.onLine只会在机器未连接到局域网或路由器时返回false，其他情况下均返回true。
也就是说，机器连接上路由器后，即使这个路由器没联通网络，navigator.onLine仍然返回true。
